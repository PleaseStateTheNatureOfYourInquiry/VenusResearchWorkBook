Editing

If editing this document, to mark an image as good with a green square () add |good|, mark bad with a red square () add |bad|, mark that imshow does not work with a blue square () add |nodisp| and unknown/not tested with a white sqaure () add |none| before “Image” like so:

|good| Image: image.img
|bad| Image: image.img
|none| Image: image.img
|nodisp| Image: image.img

See Usage for documentation on how to open the images.
